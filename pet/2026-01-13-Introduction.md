# PET基本介绍

PET检查，全称为**正电子发射断层成像术**（**P**ositron **E**mission **T**omography），是目前医学影像领域最先进的**分子显像**技术之一。

如果说CT和MRI主要看器官的“长相”（解剖结构），那么PET则主要看器官的“工作状态”（代谢功能）。它在**肿瘤早期诊断、精准分期和疗效评估**方面具有不可替代的优势，被誉为“探测肿瘤的雷达”。

**🔬 基本原理：捕捉能量的“碰撞”**

PET检查的核心在于利用人体细胞对特定能量（通常是葡萄糖）的代谢差异。

1. **注射示踪剂：** 最常用的药物是 $^{18}\text{F-FDG}$（脱氧氟葡萄糖）。由于肿瘤细胞生长迅速，它们会“贪婪”地吸收这些带有放射性标记的葡萄糖。  
2. **正电子湮灭（Annihilation）：** 示踪剂在体内释放出**正电子**，正电子与体内的电子相遇并发生“湮灭”，产生两束方向相反、能量均为 $511\text{ keV}$ 的**伽马射线**。  
3. **对向探测：** PET仪器的探测环会精准捕捉到这两束同时产生的射线。  
4. **代谢显像：** 计算机根据射线产生的密度和位置，还原出人体各部位的代谢强度。代谢异常活跃的地方（如肿瘤）在图像上会显示为明显的“高亮”区域。

**🩺 主要应用领域**

PET通常与CT结合使用（即 **PET-CT**），实现功能与解剖的完美融合：

* **恶性肿瘤（核心应用）：**  
  * **早期发现：** 在器官结构尚未发生明显变化时，捕捉到代谢异常，从而发现早期肿瘤。  
  * **良恶性鉴别：** 辅助判断结节或肿块的性质。  
  * **精准分期：** 全身扫描，查看肿瘤是否有远程转移。  
  * **疗效评估：** 在化疗或放疗后，观察肿瘤代谢是否下降，判断治疗是否有效。  
* **神经系统疾病：**  
  * 辅助诊断**阿尔茨海默病**（老年痴呆）等变性疾病。  
  * **癫痫**灶的准确定位。  
* **心脏疾病：**  
  * 评估**心肌存活情况**，是决定是否进行冠脉搭桥或支架手术的“金标准”。

**🆚 PET 与 SPECT 的区别**

虽然两者都属于核医学检查，但有明显不同：

| 特性 | SPECT | PET |
| :---- | :---- | :---- |
| **放射源** | 单光子放射性核素 | 正电子放射性核素 |
| **分辨率** | 较低（约 7-10mm） | **较高**（约 4-6mm），图像更清晰 |
| **灵敏度** | 较低 | **极高**，能发现更微小的病灶 |
| **常用药物** | 锝-99m等 | 氟-18 (FDG) 等 |
| **主要用途** | 骨扫描、肾功能、甲状腺 | **肿瘤筛查、脑代谢、心肌存活** |

**📝 检查流程与重要注意事项**

### **1\. 检查前准备**

* **严格禁食：** 检查前需**禁食 4-6 小时**（通常只能喝白开水）。这是为了降低体内血糖水平，让肿瘤细胞在“饥饿”状态下充分吸收放射性葡萄糖。  
* **血糖控制：** 糖尿病患者需提前告知医生，将血糖控制在一定范围内，否则会影响显像效果。  
* **静卧休息：** 注射药水后需在安静、避光的房间内**静卧约 1 小时**，避免说话或走动，防止药物进入肌肉组织影响结果。

### **2\. 检查后护理**

* **大量饮水：** 检查结束后应多喝水，通过尿液尽快将体内的放射性药物排出。  
* **短期隔离：** 药水的半衰期很短（约 110 分钟），但在检查后的 **24 小时内**，建议避免近距离接触孕妇和婴幼儿。

---

**温馨提示：** PET-CT检查虽然非常强大，但并非所有人都需要作为常规体检项目。它通常用于高危人群筛查或已确诊患者的深度评估。
