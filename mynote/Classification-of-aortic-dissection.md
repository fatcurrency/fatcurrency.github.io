# 主动脉夹层的分型及治疗原则

## 1. 主动脉夹层的分型

传统分型中应用最广泛的是 **Debakey分型** 和 **Stanford分型**。其中 **Stanford分型更为简洁实用**。

### 1.1 Debakey 分型

根据夹层起源和扩展范围分为三型：

| 分型 | 描述 |
| :--- | :--- |
| **I 型** | 胸主动脉夹层起源于升主动脉，并向远端扩展，累及腹主动脉。 |
| **II 型** | 胸主动脉夹层局限于升主动脉。 |
| **III 型** | 胸主动脉夹层起源于胸降主动脉。 |
|  - IIIA型 | 向下未累及腹主动脉。 |
|  - IIIB型 | 向下累及腹主动脉。 |

### 1.2 Stanford 分型

根据是否累及升主动脉分为两型：

| 分型 | 描述 | 对应 Debakey 分型 |
| :--- | :--- | :--- |
| **A 型** | 无论夹层起源于何处，只要**累及升主动脉**。 | I 型、II 型 |
| **B 型** | 夹层**起源于胸降主动脉且未累及升主动脉**。 | III 型 |

## 2. 治疗原则

采取以**手术治疗为主**的综合治疗方式。

### 2.1 急性期综合治疗（基础治疗）

- **目标**： 控制疼痛、血压和心率，防止夹层进一步扩大。
- **措施**：
    - **充分镇痛**。
    - **控制血压**： 收缩压降至 **100~120 mmHg**。
    - **控制心率**： 心率降至 **60~80 次/分**。
    - **药物**： 如无禁忌，均应使用 **β受体阻滞剂**。
    - **监测**： 注意观察肾脏灌注（尿量应避免 < 25 ml/h）。

- **禁忌药物**： **严禁使用**溶栓剂、抗凝剂、抗血小板药物。

### 2.2 后期手术治疗

根据分型选择不同手术方案：

| 分型 | 首选治疗方案 | 说明 |
| :--- | :--- | :--- |
| **Stanford A 型** | **外科开放手术** | 急、慢性A型夹层最有效的治疗，手术效果明显优于内科保守治疗。 |
| **Stanford B 型** | **主动脉腔内隔绝术（介入治疗）** | 主要适用于B型夹层。 |
