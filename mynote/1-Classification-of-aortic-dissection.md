# 主动脉夹层的分型及治疗原则

主动脉夹层（aortic dissection，AD）也称为**主动脉夹层动脉瘤**，是一种严重的心血管急症。当动脉壁内膜出现破口，血液通过破口进入动脉壁形成血肿，并进一步剥离主动脉的内膜和中膜，便会产生主动脉夹层。公认的发病原因是**主动脉中膜本身的结构异常和血流动力学异常**导致的，且发病高危人群包括高血压病、年龄较大、主动脉粥样硬化和遗传性血管病患者等。此病病情进展迅速，早期死亡率高，且发病率与人口年龄增长有关。病人大多会有<span style="color: #ff3814">突发剧烈胸痛</span>等症状，并可能出现<span style="color: #ff3814">面色苍白、出汗、四肢皮肤湿冷和灌注不良等类似休克的症状</span>，部分患者可<span style="color: #ff3814">有晕厥或意识障碍</span>等经常与其他疾病相混淆，因此需要对可能患者进行详细的病史询问和体格检查。

对于主动脉夹层的治疗，首先需要进行**有效的镇痛和控制血压和心率**，降低主动脉的破裂风险。根据不同的疾病状态和类型，一些病情严重的患者需要积极的手术治疗以预防主动脉破裂和心脏压塞等事故发生，而对于病情较轻的患者可选择药物战术进行治疗。

## 1. 主动脉夹层的分型

传统分型中应用最广泛的是 **Debakey分型** 和 **Stanford分型**。其中 **Stanford分型更为简洁实用**。

### 1.1 Debakey 分型

根据夹层起源和扩展范围分为三型：

| 分型 | 描述 |
| :--- | :--- |
| **I 型** | 胸主动脉夹层起源于升主动脉，并向远端扩展，累及腹主动脉。 |
| **II 型** | 胸主动脉夹层局限于升主动脉。 |
| **III 型** | 胸主动脉夹层起源于胸降主动脉。 |
|  - IIIA型 | 向下未累及腹主动脉。 |
|  - IIIB型 | 向下累及腹主动脉。 |

### 1.2 Stanford 分型

根据是否累及升主动脉分为两型：

| 分型 | 描述 | 对应 Debakey 分型 |
| :--- | :--- | :--- |
| **A 型** | 无论夹层起源于何处，只要**累及升主动脉**。 | I 型、II 型 |
| **B 型** | 夹层**起源于胸降主动脉且未累及升主动脉**。 | III 型 |

## 2. 治疗原则

采取以**手术治疗为主**的综合治疗方式。

### 2.1 急性期综合治疗（基础治疗）

- **目标**： 控制疼痛、血压和心率，防止夹层进一步扩大。
- **措施**：
    - **充分镇痛**。
    - **控制血压**： 收缩压降至 **100~120 mmHg**。
    - **控制心率**： 心率降至 **60~80 次/分**。
    - **药物**： 如无禁忌，均应使用 **β受体阻滞剂**。
    - **监测**： 注意观察肾脏灌注（尿量应避免 < 25 ml/h）。

<span style="color: #ff3814">- **禁忌药物**： **严禁使用**溶栓剂、抗凝剂、抗血小板药物。</span>

### 2.2 后期手术治疗

根据分型选择不同手术方案：

| 分型 | 首选治疗方案 | 说明 |
| :--- | :--- | :--- |
| **Stanford A 型** | **外科开放手术** | 急、慢性A型夹层最有效的治疗，手术效果明显优于内科保守治疗。 |
| **Stanford B 型** | **主动脉腔内隔绝术（介入治疗）** | 主要适用于B型夹层。 |
